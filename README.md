# Localization and Hangfire / Cron Jobs Projects

This repository contains .NET projects focused on **localization** and **scheduled background jobs** using Hangfire.

## Tech Stack
- .NET 6 / .NET 7
- C# / ASP.NET Core
- Hangfire for Cron jobs and background processing
- Localization with .resx files

## Projects Included
- LocalizationProject — Adding multilingual support to projects using .resx
- CustomeHangfireClean — Custom Hangfire background job implementations
- CustomeHangfireCleanCopyforDocker — Hangfire jobs setup for Docker environments
- CustomeHangfireJob — Custom Hangfire job experimentation
- HangfireCronDotNet9 — Cron jobs using Hangfire in .NET 9

## Notes
- Secrets (API keys, tokens, database credentials) should **never be committed**.
- Projects are primarily demo / learning projects for Hangfire scheduling and localization implementation.
